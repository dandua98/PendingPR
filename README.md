# PendingPR

> A GitHub App built with [Probot](https://github.com/probot/probot) to help you manage issues that are in pull requests better

## Setup

```sh
# Install dependencies
yarn install

# Compile TypeScript
yarn build

# Run the bot
yarn start
```
